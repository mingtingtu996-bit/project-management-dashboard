## 🔍 常见问题排查

### Q1: 编译失败 - "Cannot find module 'ConditionsManagement'"
**A**: 检查文件路径是否正确
```bash
ls client/src/pages/ConditionsManagement.tsx
```
文件应该存在

### Q2: 编译失败 - "Cannot find module 'ListChecks'"
**A**: Lucide-react 中可能没有此图标，改用其他图标
```typescript
// 可选图标：
CheckList, ListTodo, Clipboard, ClipboardList, CheckSquare
```

### Q3: 菜单上看不到"条件管理"
**A**: 清除浏览器缓存或进行 hard refresh
```
Ctrl+Shift+Delete (Chrome/Edge)
Cmd+Shift+Delete (Mac Chrome)
```

### Q4: 点击"条件管理"菜单项但页面不加载
**A**: 检查路由是否正确配置
```
URL应该是: http://localhost:5173/#/projects/PROJECT_ID/conditions
```

### Q5: 页面加载很慢
**A**: 这是正常的懒加载行为，第一次打开会加载新的模块
- 刷新一次后应该会快速加载
