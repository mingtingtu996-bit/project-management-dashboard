# 前期证照交互规则稿_20260409

> 适用范围：`证照管理 / 前期证照`
>
> 文档定位：用于统一前期证照页的筛选、高亮、抽屉、共享事项条带和升级动作的交互规则。
> 导航口径说明：侧边栏父级最终显示名统一为 `专项管理`；本文中保留的 `证照管理` 仅用于既有专项命名和技术语境。


## 1. 总体原则

- 首页先看结果，再看推进，再看细项
- 不做复杂关系图
- 所有深层信息统一进入抽屉
- 共享事项只做轻量高亮，不做满屏连线

## 2. 总览区交互

- 点击 `当前卡点` 卡片：
  - 自动筛选到对应证件
  - 看板高亮该证件
- 点击 `逾期事项`：
  - 台账自动切换到仅看逾期
- 点击 `预计具备开工条件日期`：
  - 弹出影响开工条件的关键事项列表

## 3. 四证看板交互

- 点击任一证件行：
  - 打开 `CertificateDetailDrawer`
- Hover 任一证件行：
  - 同步高亮与之相关的共享事项条带
- 点击阻塞标签：
  - 抽屉直接定位到阻塞记录区块

## 4. 共享事项条带交互

- 点击条带：
  - 相关证件行高亮
  - 台账自动过滤到该共享事项相关记录
- Hover 条带：
  - 非相关证件弱化

## 5. 台账交互

- 默认按状态 + 计划日期排序
- `新增办理事项` 打开表单弹窗
- `查看详情` 打开抽屉
- `升级为问题` 和 `升级为风险` 成功后：
  - 当前行显示已关联标记
  - 抽屉联动刷新

## 6. 抽屉交互

抽屉内固定四个主区块：

- 基本信息
- 共享事项与依赖
- 状态记录
- 问题与联动

抽屉内部按钮：

- `查看共享事项`
  - 滚动到共享事项区
- `查看依赖矩阵`
  - 打开矩阵弹层或展开区
  - 矩阵行列定义：行为四证（土地证 / 用地规划许可证 / 工程规划许可证 / 施工许可证），列为共享办理事项
  - 单元格状态色：✓ 绿底（已满足）/ ○ 蓝底（待办）/ ✗ 红底（阻塞）/ — 灰底（无关）
  - 点击单元格：高亮看板中对应的证件行与事项条带，不打开新弹窗
  - 矩阵为只读展示，编辑入口在台账和抽屉
- `升级为问题`
- `升级为风险`

## 7. 筛选规则

固定筛选项：

- `仅看阻塞`
- `仅看逾期`
- `仅看补正`
- `按证件类型`

筛选联动规则：

- 看板筛选与台账筛选保持同步
- 抽屉保持当前选中对象，不因为筛选直接关闭

## 8. 移动端规则

- 默认先进入台账
- 看板折叠为单证件卡片列表
- 抽屉改全屏弹层

## 9. 禁止事项

- 不允许共享事项条带引出复杂关系图
- 不允许在首页平铺全量记录
- 不允许一键升级后不回写页面标记
