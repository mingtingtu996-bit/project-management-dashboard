# 施工图纸测试与验收清单_20260409

> 适用范围：`证照管理 / 施工图纸`
> 导航口径说明：侧边栏父级最终显示名统一为 `专项管理`；本文中保留的 `证照管理` 仅用于既有专项命名和技术语境。


## 1. 页面结构验收

- `Drawings` 路由可达
- `Sidebar` 中存在 `施工图纸`
- `PreMilestones` 不再承载完整图纸主视图

## 2. 数据口径验收

- 图纸包首页显示的是包级对象，不是单图列表
- 应有项缺漏和齐套度计算正确
- 当前有效版唯一
- `review_mode` 显示正确

## 3. 交互验收

- `新建图纸包` 可生成模板
- `查看缺漏` 可正确筛出缺失项
- `查看版本` 可打开版本窗口
- `设为当前有效版` 后历史版本弱化

## 4. 联动验收

- 施工执行类图纸包可映射任务条件
- 竣工归档类图纸包可映射验收前置
- 自动识别的问题可升级为问题 / 风险

## 5. 回归验收

- 旧 `construction_drawings` 数据仍可被新页面读取
- 图纸创建、更新、删除后首页摘要正确刷新

- `src/pages/__tests__/DrawingsBoard.test.tsx`
- `src/pages/__tests__/DrawingsLedger.test.tsx`
- `server/src/__tests__/drawingPackageService.test.ts`
