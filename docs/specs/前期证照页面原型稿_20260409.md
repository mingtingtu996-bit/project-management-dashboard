# 前期证照页面原型稿_20260409

> 适用范围：`证照管理 / 前期证照`
>
> 文档定位：承接 `前期证照产品改造稿_20260408.md`，用于统一页面布局、按钮位置、视觉层级和与当前双 tab 页的迁移方式。
> 导航口径说明：侧边栏父级最终显示名统一为 `专项管理`；本文中保留的 `证照管理` 仅用于既有专项命名和技术语境。


## 1. 页面目标

前期证照页要回答的不是“资料都有哪些”，而是：

- 四证目前推进到哪一步
- 哪个证件或共享事项卡住开工
- 预计何时具备开工条件
- 当前最需要处理的办理事项是什么

因此页面的核心结构应是：

1. 开工准入总览
2. 四证推进看板
3. 办理台账
4. 详情抽屉

## 2. 与当前页面的迁移关系

当前 `PreMilestones` 页面存在两个 tab：

- `前期证照`
- `施工图纸`

目标状态必须明确为：

- `PreMilestones` 只保留 `前期证照`
- 当前 `施工图纸` tab 从页面内部移出
- `施工图纸` 改为 `证照管理` 下的平级子页 `/projects/:id/drawings`
- 前期证照页内如需保留图纸入口，只允许用摘要跳转，不再保留并列 tab 主体

结论：

- 后续实现时，不允许把双 tab 结构继续当长期方案保留

## 3. 页面信息架构

## 3.2 顶部总览

顶部只展示结果，不堆细节。

建议卡片：

- `四证完成`
- `当前卡点`
- `预计具备开工条件日期`
- `逾期事项`
- `待补正事项`
- `本周推进事项`

卡片下方允许显示一行辅助说明，例如：

- `施工许可证仍受共享事项阻塞`
- `当前最慢证件：工程规划许可证`

## 3.3 共享事项条带

共享事项条带是首页唯一的轻量关系表达。

展示规则：

- 一个共享事项显示为一条横向条带
- 条带跨越它影响的证件行
- 同一共享事项使用统一颜色标识
- 点击条带时，相关证件行同步高亮，其余弱化

条带信息不宜过多，建议只展示：

- 事项名称
- 当前状态
- 影响证件数
- 是否阻塞

## 3.4 四证推进看板

固定四行：

- 土地证
- 用地规划许可证
- 工程规划许可证
- 施工许可证

每行固定四段阶段：

- `资料准备`
- `内部报审`
- `外部报批`
- `批复领证`

每行只展示高频信息：

- 当前阶段
- 当前状态
- 审批部门
- 下一动作
- 阻塞提示

不在行内展示：

- 长备注
- 全量依赖列表
- 全量资料列表

## 4. 办理台账

台账是主编辑入口。

推荐列：

- 办理事项
- 所属证件
- 当前阶段
- 当前状态
- 计划完成日期
- 审批部门
- 是否补正
- 是否阻塞
- 操作

推荐操作区按钮：

- `新增办理事项`
- `编辑`
- `查看详情`
- `升级为问题`
- `升级为风险`

## 5. 详情抽屉

详情抽屉统一承接单项处理。

建议分 4 个区块：

1. 基本信息
2. 共享事项与依赖
3. 状态记录
4. 问题与联动

### 5.1 基本信息

展示：

- 证件名称
- 当前状态
- 当前阶段
- 审批部门
- 计划 / 实际日期
- 下一动作

### 5.2 共享事项与依赖

展示：

- 受哪些共享事项影响
- 当前事项还影响哪些证件
- 轻量依赖矩阵

操作按钮：

- `查看共享事项`
- `查看依赖矩阵`

### 5.3 状态记录

展示：

- 最近状态变更
- 最近补正记录
- 最近阻塞记录
- 更新时间

### 5.4 问题与联动

展示：

- 预警状态
- 是否已升级为问题
- 是否已升级为风险

操作按钮：

- `升级为问题`
- `升级为风险`

## 6. 视觉统一要求

- 页面壳统一使用当前系统 `PageHeader + Breadcrumb + shell-surface`
- 顶部摘要卡延续现有 `Dashboard` 和 `RiskManagement` 的白底卡片语言
- 四证行卡以白底、浅边框、蓝色或橙色小标签表达状态，不使用重色大块覆盖
- 共享事项条带允许有轻强调色，但透明度要低，不能压过四证行本身
- 抽屉继续沿用当前 `Dialog / Drawer` 风格，不另起一套证照主题

## 7. 按钮清单

- 页面级按钮
  - `查看施工图纸`
  - `查看全部逾期`
  - `新增办理事项`
- 看板内按钮
  - `查看详情`
- 抽屉内按钮
  - `查看共享事项`
  - `查看依赖矩阵`
  - `升级为问题`
  - `升级为风险`

## 8. 移动端降级

- 默认进入台账视图
- 四证看板改成纵向卡片堆叠
- 共享事项条带降级为列表摘要
- 详情仍走抽屉或全屏弹层
