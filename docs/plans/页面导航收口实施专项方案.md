- 正文任一步骤开始前，必须先对照《三份专项方案执行前检查清单.md》筛出本步适用的 `blocked` 条件；若命中任一阻断项，当前步骤不得直接进入实现。

- 页面壳层 / 组件步骤统一采用 `contract-test -> impl -> review`；纯视觉一致性和命名残留复核统一采用 `story/e2e -> review`。
