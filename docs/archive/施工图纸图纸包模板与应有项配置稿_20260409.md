# 施工图纸图纸包模板与应有项配置稿_20260409

> 适用范围：`证照管理 / 施工图纸`

## 1. 目标

本稿定义：

- 图纸包模板怎么建
- 应有项怎么生成
- 哪些包默认必审

## 2. 模板对象

每个模板至少包含：

- `template_code`
- `template_name`
- `discipline_type`
- `document_purpose`
- `default_review_mode`
- `items`

## 3. 模板最小集合

建议首批模板：

- 建筑施工图包
- 结构施工图包
- 给排水施工图包
- 暖通施工图包
- 电气施工图包
- 消防专项包
- 人防专项包
- 竣工归档包

## 4. 应有项最小字段

- `item_code`
- `item_name`
- `is_required`
- `sort_order`

## 5. 示例

### 5.1 建筑施工图包

- 平面图
- 立面图
- 剖面图
- 详图

### 5.2 结构施工图包

- 基础图
- 梁板配筋图
- 柱墙配筋图
- 节点详图

### 5.3 竣工归档包

- 竣工总说明
- 竣工图目录
- 各专业竣工图

## 6. 默认送审规则

建议首批默认：

- `消防专项包` -> `mandatory`
- `人防专项包` -> `mandatory`
- `竣工归档包` -> `manual_confirm`
- 常规施工执行包 -> `none`

## 7. 项目级修正规则

- 项目可删除不适用应有项
- 项目可新增特有项
- 项目可覆盖 `review_mode`

## 8. 页面承接

- `新建图纸包` 时优先从模板生成应有项
- 首页的缺漏/齐套度直接以模板应有项为基底计算
